# WebLab: Аналіз експресії генів і heatmap
**Live Demo:** [baza17.bezpekaservice.com.ua](https://baza17.bezpekaservice.com.ua/)
---
## Використані технології

* **Backend:** PHP (для обробки серверних запитів та динамічного завантаження даних генів) .
* **Frontend:** HTML5, CSS3, JavaScript.
* **Асинхронна взаємодія:** **HTMX** (для швидких запитів до бази даних у реальному часі) та **AJAX** (сучасна відправка інтерактивних форм).
* **Стилізація та UI:** **Bootstrap 5**, інтеграція та адаптація преміум-шаблону **Grayscale** під біоінформатичну тематику.

---

## Основні сторінки та функціонал

* **Головна:** Анотація сайту, інтерактивна панель управління; реалізовано динамічний запит до бази даних за допомогою **HTMX** та контактна форма з **AJAX-відправкою** (без перезавантаження сторінки).
* **Суть:** Теоретична основа аналізу експресії генів для ознайомлення користувача з предметною областю.
* **Heatmap:** Інтерактивний модуль теплової карти. Демонструє роботу з демо-матрицею даних (включаючи математичну нормалізацію значень) та містить детальні пояснення, як правильно інтерпретувати та читати теплові карти.
* **Up/Down:** Освітній розділ, що пояснює біологічну різницю між *Up-regulated* (активованими) та *Down-regulated* (пригніченими) генами.
* **Приклад:** Практичне використання теплових карт у реальних наукових дослідженнях. Містить:
  * Інтерактивну фотогалерею **DEG analysis** (диференціально експресованих генів), де три ілюстрації та їхні підписи плавно і синхронно змінюються при кліку.
  * Список використаних наукових джерел із клікабельними посиланнями.
* **Bootstrap:** Сторінка поширених запитань (FAQ), побудована на компонентах Bootstrap (Accordion), а також додатковий **HTMX-запит** до бази даних.
* **Bootstrap 2:** Повна адаптація та стилізація адаптивного шаблону *Grayscale* під специфіку аналізу генетичних даних.

## Структура проекту 

```text
WebLab/
├── 📁 .vscode/
│   └── 📄 launch.json                 # Конфігурація середовища VS Code для налагодження PHP
├── 📁 assets/                         # Статичні ресурси додатку
│   ├── 📁 images/                     # Графічні матеріали (ілюстрації)
│   ├── 📄 app.js                      # Головний скрипт ініціалізації додатку
│   ├── 📄 form.js                     # Обробка валідації та AJAX-відправки форм
│   ├── 📄 gallery.js                  # Скрипт плавного керування інтерактивною галереєю DEG
│   └── 📄 styles.css                  # Кастомні стилі вашого проєкту та темної теми
├── 📁 grayscale/                      # Оригінальні файли та ресурси шаблону Grayscale
├── 📄 bootstrap-page.php              # Сторінка FAQ на Bootstrap з вбудованими HTMX-запитами
├── 📄 db.php                          # Скрипт підключення та взаємодії з базою даних
├── 📄 essence.html                    # Теоретичний розділ: Суть аналізу експресії генів
├── 📄 example.html                    # Практичні приклади використання теплових карт та джерела
├── 📄 grayscale.php                   # Повна адаптація теми Grayscale під аналіз експресії генів
├── 📄 heatmap.html                    # Інтерактивна теплова карта та демонстрація нормалізації матриці
├── 📄 index.php                       # Головна сторінка, анотація проєкту та панель управління
├── 📄 load_genes.php                  # Серверний PHP-скрипт динамічного завантаження даних експресії
├── 📄 load_genes_bootstrap.php        # Бекенд-компонент для інтеграції даних із Bootstrap-інтерфейсом
├── 📄 submit_form.php                 # PHP-обробник для AJAX-відправки даних із контактної форми
├── 📄 updown.html                     # Теоретичний розділ: Детальне пояснення Up- та Down-regulated генів
└── 📄 README.md                       # Документація проєкту
```
